Fluento — AI-Powered English Speaking Coach
Fluento is a full-stack AI application that provides personalised English speaking practice with real-time evaluation. The system is built around a complete speech processing pipeline — user audio is recorded via the browser, transcribed locally using faster-whisper (OpenAI Whisper model running on CPU, no external API cost), then evaluated by LLaMA 3.3 70B via Groq API which returns structured JSON containing fluency, grammar, vocabulary, clarity and pacing scores, specific corrections with rewrites, and a full corrected version of the speech.
The application is fully goal-aware — users set their speaking goal (job interviews, IELTS, business English) and difficulty level during onboarding, and every AI generation in the app adapts to that context. Topics, vocabulary words, reading paragraphs and structure hints are all generated dynamically using prompt-engineered LLM calls rather than hardcoded content.
Key technical implementations include a dual-cursor reading mode built on the Web Speech API with a high-water mark ratchet to prevent cursor regression during speech engine restarts, a lookahead word matching algorithm for real-time skip detection, and a modular FastAPI backend with SQLAlchemy ORM handling session persistence, streak calculation and analytics aggregation.
Tech Stack: FastAPI · Python · faster-whisper · Groq API · LLaMA 3.3 70B · Next.js 16 · TypeScript · Framer Motion · Web Speech API · SQLAlchemy · SQLite
